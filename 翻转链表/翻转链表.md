# 翻转链表

标签（空格分隔）： 未分类

---

### 题目描述：
输入一个链表，反转链表后，输出链表的所有元素

### 思路：
借用三个指针，pre，head，next，画图会非常显著，在此不赘述

#### 代码：
```
class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}

public class Solution {

    public ListNode ReverseList(ListNode head) {

        if (head == null || head.next == null)
            return head;

        ListNode pre = null;
        ListNode next = null;

        while (head != null) {

            next = head.next;
            head.next = pre;
            pre = head;
            head = next;
        }
        return pre;

    }
}
```
